ABOUTME

  • HELLO!

    웹 콘텐츠를 구현하는 것을 보람으로 삼는 웹 프론트엔드 개발자 이준영입니다. 웹에 대해 공부한 이후 웹은 서핑하도록 단순히 꾸며놓은 곳이 아닌 여러 가지 언어로 사용자와 그들의 편의를 고려한 잘 짜여진 하나의 세계라고 여기게 되었습니다. 현재 목표는 많은 사용자들이 만족할 수 있는 UX UI를 제공하는 것이며 앞으로의 목표는 전문적인 풀 스택 웹 개발자가 되는 것입니다.

  • WHAT I DO?

    01 HTML AND CSS

    HTML 시멘틱 마크업에 대해 이해하고 있으며 디자인된 화면을 CSS로 표현 가능하고 동적인 소스를 활용할 수 있습니다.

    02 JAVASCRIPT

    기존에 있던 코드의 작동 원리를 이해하며 상황에 맞게 수정하고 변수, 함수, 메소드, 프로토타입을 다양하게 활용하여 코드를 직접 작성할 수 있습니다.

    03 RESPONSIVE

    모바일, 태블릿, PC 어디서나 최적화된 사이즈로 웹 사이트의 접속이 가능하도록 제작할 수 있습니다.

    04 CROSS BROWSING

    IE에 국한되지 않은 크롬, 사파리, 파이어폭스 등과 호환되도록 웹 표준 방식으로 제작할 수 있습니다.
